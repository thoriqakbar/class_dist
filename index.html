One-time Firebase setup required
  1. Go to console.firebase.google.com → Add project (free Spark plan).
  2. In the project: Build → Realtime Database → Create database → choose Start in test mode.
  3. Click the gear → Project settings → scroll to Your apps → </> Web → register app → copy the firebaseConfig object.
  4. Open index.html in a text editor and replace the PASTE_YOUR_… placeholders near the top of the <script> section.
  5. Push to GitHub Pages — done!
